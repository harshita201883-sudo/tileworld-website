## 🧱 Overview
Elite Tiles is a modern, fully responsive website designed to showcase high-quality tiles for both residential and commercial spaces.  
It combines elegant design, smooth animations, and strong performance — built for a premium user experience.

---

## ✨ Features
- 🎥 **Video Background Hero Section** – Stylish entry with smooth text overlay  
- 📱 **Responsive Navbar** – Auto transforms into a mobile-friendly hamburger menu  
- 🧩 **Product Collections** – Modern tile designs displayed in a grid layout  
- 🗺️ **Map & Contact Form** – Easy access for customer inquiries  
- 🏆 **Why Choose Us Section** – Highlighting quality, trust, and innovation  
- 🎨 **Fully Responsive Design** – Works beautifully on desktop and mobile  

---

## 🛠️ Technologies Used
- **HTML5**  
- **CSS3 (Flexbox + Grid)**  
- **JavaScript (Vanilla JS)**  
- **Font Awesome**  
- **Pexels Video Backgrounds**
